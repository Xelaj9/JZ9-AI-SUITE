# JZ9 AI Suite

3-in-1 AI tools powered by OpenAI:
- 🧬 Visual DNA Extractor (GPT-4o Vision)
- ⌨️ Prompt Editor (GPT-4o-mini)
- 🎨 Graphic Engine (GPT-4o + GPT-Image-1)

## Setup

1. Push to GitHub
2. Import to Vercel
3. Add Environment Variable:
   - Key: `OPENAI_API_KEY`
   - Value: `sk-...`
4. Redeploy

## Files
- `index.html` — Frontend (all 3 tools)
- `api/proxy.js` — OpenAI proxy
- `vercel.json` — Routing config
