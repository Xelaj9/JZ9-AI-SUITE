Features

ทุกอย่างสำหรับงาน Prompt ของคุณ

ตั้งแต่เก็บไอเดีย ไปจนถึงส่งงานจริง

📚

Prompt Library ส่วนตัว

เก็บ Prompt ทั้งหมดไว้ที่เดียว จัดหมวดหมู่ ติด tag และ favorite ที่ใช้บ่อย

🖼

แนบรูปภาพกับ Prompt

อัปโหลดรูปอ้างอิง เก็บใน gallery ส่วนตัว แล้วแนบกับ Prompt ได้หลายรูป

🔍

ค้นหาและจัดหมวดหมู่

ค้นหาเร็ว กรองตามประเภทงาน/tag ทำสำเนาเพื่อปรับต่อได้ในคลิกเดียว

🎨

เชื่อมกับ Studio

สร้าง Prompt จากเครื่องมือ AI แล้วกด Save to Library ได้ทันที

🧰

Image Tools ฟรี

ดึงโค้ดสี แปลงไฟล์ เช็ค Safe-Zone ใส่ลายน้ำ หั่นภาพลง IG — ทำในเบราว์เซอร์ ไม่ต้องอัปโหลดขึ้นเซิร์ฟเวอร์

🔒

ข้อมูลเป็นส่วนตัว
